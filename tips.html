Cyber Security Tips

Use Strong Passwords

Create passwords with at least 12 characters, including uppercase letters, numbers and symbols.

Enable 2FA

Use Two-Factor Authentication to add an extra layer of security to your accounts.

Beware of Phishing

Never click suspicious links or download attachments from unknown senders.

Secure Public Wi-Fi

Avoid accessing sensitive accounts on public networks without protection.

Keep Software Updated

Install security updates regularly to protect against newly discovered vulnerabilities.

Protect Personal Data

Limit the amount of personal information shared on social media and public platforms.

Quick Security Checklist

Password Manager

Store passwords securely and avoid reusing them.

Backup Data

Keep important files backed up regularly.

Verify Websites

Check HTTPS and website legitimacy before entering information.
